Instrucciones

El archivo CSV debe tener exactamente estas columnas en la primera fila:

nombre, unidad, stock_actual, stock_minimo
  • nombre: texto sin números (ej: Harina de trigo)
  • unidad: g (gramos) o ml (mililitros)
  • stock_actual y stock_minimo: números decimales (ej: 1500.5)

Si un insumo con el mismo nombre ya existe, se actualiza. Si no existe, se crea.
